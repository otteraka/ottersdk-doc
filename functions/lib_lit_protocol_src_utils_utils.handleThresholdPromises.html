Function handleThresholdPromises

  • handleThresholdPromises(promisesArray, threshold): Promise<any[]>

  • Handles an array of asynchronous connection and ensures that a specified minimum number of these connections are successfully established. This function is crucial in decentralized or distributed systems, such as the Lit Protocol networks, where establishing connections to multiple nodes can enhance reliability and resilience of the network operations.

    Each promise in the input array attempts to connect to a specified network address and may resolve with some result data on success or reject on failure. This function aggregates these promises and tracks their outcomes, resolving its own returned promise only if at least a specified threshold of connections succeed. This mechanism allows the system to tolerate a degree of failure typical in large, distributed networks without compromising the overall operation.

    If the threshold is not met by the time all promises are settled, the function rejects its promise indicating insufficient successful connections, which can trigger error handling strategies or retries in the consuming system.

    Parameters

    • promisesArray: ResponsePromise[]

      An array of objects, each object having a network address as a key and a promise as its value. Each promise represents an asynchronous attempt to establish a connection to the network node at the corresponding address.

    • threshold: number

      The minimum number of successful connections required for the function to resolve its returned promise successfully.

    Returns Promise<any[]>

    A promise that resolves with an array of results from successful connections once the required threshold is met. Each result object contains networkAddress and the data from the successful connection. If the threshold is not met, the promise rejects with an error.

    Example

    handleThresholdPromises(responsePromises, 3)
      .then(results => {
        console.log("Successfully connected to nodes:", results);
      })
      .catch(error => {
        console.error("Failed to meet the threshold for successful connections:", error);
      });
